Passkeys

Passkeys are a secure way to authenticate sensitive actions. Janus uses them as an extra layer of security before allowing you to access AWS credentials or the AWS console. You need at least one passkey to access AWS, but we recommend registering two.

A portable passkey gives you resilience and lets you sign in from other devices.

Titan security keys are available from the IT service desk in any office. This is the preferred and best-supported option.

If you use a phone, the Microsoft Authenticator app will not work as it can only create passkeys for Microsoft accounts. Instead, use your phone's built-in passkey storage, such as Apple Passwords (iPhone) or Google Password Manager (Android).

An on-device passkey is the fastest and simplest option for everyday use.
fingerprint

On a Mac, register a passkey in the Chrome browser to use Touch ID.

On Windows, a fingerprint passkey may not be available, as Windows Hello is disabled by default. Use a portable passkey instead.

Firefox doesn't yet have its own passkey-management system, so you'll either need to use Google Chrome for Janus or stick to using a portable passkey.
