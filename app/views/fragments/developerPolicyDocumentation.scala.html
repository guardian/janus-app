Developer policies allow teams to provision access to AWS from their own application stacks without needing to hard-code bespoke permissions in Janus' configuration. Janus discovers developer policies by looking for eligible IAM policies in each AWS account.

The Grant ID comes from the AWS policy's path, and can link this policy with a Janus user's permissions using a Developer Policy Grant. A Developer Policy Grant will allow a Janus user to access all matching developer policies.
